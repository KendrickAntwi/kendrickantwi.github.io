## Features

-Responsive Design
The website adapts to different screen sizes, making it usable on desktops, tablets, and mobile devices.

-Navigation Menu
Easy-to-use navigation bar that allows users to quickly access sections like About, Projects, Contact, and Blog.

-About Section
Provides a brief introduction about the site owner and purpose of the website.

-Projects Showcase
Displays a list of projects with descriptions and previews to highlight coding skills and experience.

-Contact Form
Allows visitors to send messages directly through the website by filling out a simple form.

-Clean and Simple UI
Designed with a minimal and user-friendly interface for better readability and user experience.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KendrickAntwi/kendrickantwi.github.io.git
   ```
  2. Navigate into the project directory: ```cd kendrickantwi.github.io```
  3. Open index.html in your web browser to view the portfolio.
